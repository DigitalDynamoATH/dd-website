# Φωτογραφίες Βραβείων

Για να εμφανιστούν οι φωτογραφίες με τα βραβεία, προσθέστε τις εικόνες σας εδώ με τα εξής ονόματα:

- `award1.jpg` - Πρώτη βράβευση από τους Αετούς της Διαφήμισης 2024
- `award2.jpg` - Δεύτερη βράβευση από τους Αετούς της Διαφήμισης 2024

**Οδηγίες:**
1. Αντιγράψτε τις φωτογραφίες σας στο folder `images/`
2. Μετονομάστε τις σε `award1.jpg` και `award2.jpg`
3. Προτιμήστε JPG format για καλύτερη συμπίεση
4. Προτεινόμενο μέγεθος: 800x1000px (portrait orientation)

Οι φωτογραφίες θα εμφανίζονται αυτόματα στο section "Επιτεύγματα & Βραβεία".
