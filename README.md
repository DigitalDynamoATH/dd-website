# Digital Dynamo - Personal Biography Website

Σελίδα βιογραφίας για τον Ιωάννη Νεκτάριο Κουνουπά, CEO της Digital Dynamo.

## Περιεχόμενα

- **index.html** - Κύριο HTML αρχείο με όλες τις sections
- **styles.css** - CSS styling με modern, responsive design
- **script.js** - JavaScript για interactions και animations

## Χαρακτηριστικά

- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Modern UI/UX με gradient effects
- ✅ Smooth scrolling navigation
- ✅ Interactive animations
- ✅ Social media links (Instagram, LinkedIn)
- ✅ SEO optimized
- ✅ Fast loading

## Sections

1. **Hero Section** - Εισαγωγή με όνομα και βασικές πληροφορίες
2. **About** - Λεπτομερής περιγραφή και ιστορία
3. **Experience** - Timeline με επαγγελματική εμπειρία
4. **Achievements** - Βραβεία και επιτεύγματα
5. **Skills** - Δεξιότητες και εξειδίκευση
6. **Education** - Εκπαίδευση
7. **Contact** - Σύνδεσμοι επικοινωνίας

## Deployment

Για να ανεβάσετε το site στο digitaldynamo.gr:

1. Upload όλα τα αρχεία στον web server σας
2. Βεβαιωθείτε ότι το `index.html` είναι στο root directory
3. Configure το domain να δείχνει στο σωστό directory

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
